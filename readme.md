# Litter Detection Project

A computer vision application for detecting and classifying litter using YOLOv8, Roboflow, and yt-dlp.

## 🚀 Features
- **Dataset Integration**: Automatic download and subsetting from Roboflow.
- **Modern Architecture**: Modular Python structure for easy extension.
- **Video Processing**: Download and run inference on YouTube videos directly via URL.
- **Easy Setup**: One-click environment setup for Windows.

## 📁 Structure
- `main.py`: Main entry point.
- `config.py`: Configuration and API keys.
- `utils/`: Core logic modules (dataset, video, yolo).
- `setup.bat`: Environment initializer.
- `start.bat`: Application runner.

## 🛠️ Quick Start
1. Run `setup.bat` to install dependencies.
2. Run `start.bat` to perform a full training and inference cycle.

For more details, see [architecture.md](architecture.md) and [installation.md](installation.md).
